How to Play Simon Says

Objective: Repeat the color sequence shown by Simon.

  • Watch the pattern carefully as colors light up
  • Tap the colors in the same order
  • Each level adds one more color to remember
  • Get it right to advance to the next level

Controls:

  • New Game: Start a fresh game
  • Show Hint: Replay the current pattern
  • Check: Verify your sequence
  • Auto Solve: Show the solution
