KAKURO

Select a cell and enter a number

How to Play Kakuro

Goal: Fill the white cells with numbers 1-9 so that each run adds up to the clue number.

Rules:

  • Each run (horizontal or vertical) must add up to its clue
  • Use only numbers 1-9
  • No number can repeat within the same run
  • Black cells separate runs
